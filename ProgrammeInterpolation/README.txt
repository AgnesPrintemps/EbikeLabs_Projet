Pour utiliser ce programme:
- créez un fichier .txt avec toutes les distances et les altitudes
  (Attention: les nombres doivent être à virgule et non pas à points)
- compilez InterpolationSpline.java
- exectutez InterpolationSpline avec:
	- le nom du fichier contenant les données
	- le nom du fichier dans lequel vous voulez afficher les pentes
	- puis deux nombres a et b tel que l'energie intantanée requise pour
	  une pente de raideur p soit égale à a*p+b

Puis, le terminal affichera l'energie requise au cours de chemin
et le fichier cible aura la pente et l'altitude à chaque point du trajet

!!! Pour l'instant, le scanner ne marche pas: il faut taper les données directement sur la console après avoir executé le ficier, et taper n'importe quelle letter quand on a fini.
